How It Works

CoverScope leverages the YouTube Data API to search for cover versions of the song you enter. It then analyzes, filters, and classifies the results to provide rich, actionable insights.

Intelligent Filtering

Automatically separates real covers from noise like official videos, lyric videos, and karaoke tracks.

Trend Analysis

Generates a live "Trend Score" and upload-frequency chart based on recency, views, and volume.

Rule-Based Classification

Uses stable, keyword-based rules to categorize covers as Vocal, Instrumental, Acoustic, or Band.

ML Clustering (t-SNE)

Performs experimental ML clustering to find "natural" groups in the data, which you can compare against the stable rules.
